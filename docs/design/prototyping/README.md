## 2. Prototyping Lifecycle
- **Drafting:** Initial concepts are stored here as blueprints.
- **Validation:** Designs are reviewed against the core architecture for consistency.
- **Graduation:** Once a prototype is finalized, its core logic is moved to the primary `design/` or `tellurion-pipeline/` branches.
