# CLI-Native LLM Utilities

These utilities are optimized for terminal interaction and serve as the backbone for custom agent scripts and expert personas.

## 1. llm (by Simon Willison)
A versatile and extensible CLI tool for interacting with multiple LLMs.
- **Key Features:** Robust plugin architecture (e.g., `llm-gemini`) and a template system for saving system prompts as reusable expert personas.
- **Use Case:** Rapidly testing different "Expert" prompts and persona variations from the terminal.

## 2. Fabric
An open-source framework focused on practical and philosophical data synthesis.
- **Key Features:** Uses "Patterns" (modular prompts) to extract wisdom, summarize technical documentation, and generate structured data from raw input.
- **Use Case:** Condensing long technical manuals into high-signal reference materials for agent skills.

## 3. Aider
A terminal-based pair programming assistant with deep file-system awareness.
- **Key Features:** Optimized for multi-turn terminal interactions and efficient management of file-system context.
- **Use Case:** Acting as a reference architecture for building agents that need to perform complex, multi-file engineering tasks.
