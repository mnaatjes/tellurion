## Docs Directory

```
├── docs/
│   ├── archive/            # Previous / Un-needed documentation
│   ├── tutorials/          # Learning-oriented: Step-by-step lessons
│   ├── how-to/             # Problem-oriented: Goal-specific guides
│   ├── reference/          # Information-oriented: API details, schemas
│   ├── explanation/        # Understanding-oriented: High-level concepts, architecture
│   ├── design/             # Architecture Decision Records (ADRs) and specs
│   └── assets/             # Images, diagrams, and static files
├── mkdocs.yml              # Configuration (for MkDocs)
└── README.md               # The "Front Door"
```
